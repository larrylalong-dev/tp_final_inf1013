# API tests

Ce dossier contient deux options pour tester le flux principal:
- `flow.http` pour le client HTTP de JetBrains
- `smoke.sh` pour un test automatisable en ligne de commande

## Flux valide
1. inscription
2. creation annonce
3. envoi message
4. lecture messages de l'annonce

## Run (script)
```bash
chmod +x api-tests/smoke.sh
./api-tests/smoke.sh
```
